Installation

  1. Download the DMG for your Mac from Releases — arm64 for Apple Silicon (M1–M4), x64 for Intel.
  2. Open the DMG and drag StackPop to your Applications folder.
  3. Open Terminal and run: xattr -cr /Applications/StackPop.app
  4. Launch StackPop from Applications. It will set up Docker containers automatically.
  5. On first launch, you'll be guided through an onboarding flow to configure integrations.

Requires macOS 12+ and a Docker runtime (Docker Desktop or Colima). StackPop will help you install one if needed.
